# Анализ пользовательского поведения Яндекс.Афиши

## Описание проекта

Комплексный анализ поведения пользователей платформы Яндекс.Афиша с целью определения факторов, влияющих на лояльность клиентов и вероятность повторных покупок. Проект включает полный цикл работы с данными: от выгрузки из базы данных до формирования бизнес-рекомендаций.

## Цели исследования

1. **Построить профили пользователей** с агрегированными признаками их поведения
2. **Выявить факторы**, влияющие на вероятность повторных покупок
3. **Проверить продуктовые гипотезы** о влиянии типа мероприятий и регионов на лояльность
4. **Дать рекомендации** по стратегии удержания клиентов

## Ключевые этапы анализа

### 1. Подготовка данных
- **Выгрузка данных**: SQL-запрос к базе данных Яндекс.Афиши
- **Конвертация валюты**: Приведение выручки KZT к RUB по актуальным курсам
- **Предобработка**: 
  - Очистка от дубликатов и выбросов
  - Оптимизация типов данных
  - Фильтрация аномальных значений

### 2. Создание профилей пользователей
Для каждого пользователя рассчитаны:
- Даты первого и последнего заказа
- Характеристики первого заказа (устройство, регион, партнер, жанр)
- Количественные метрики (общее количество заказов, средняя выручка, среднее количество билетов, средний интервал между заказами)
- Бинарные признаки (is_two, is_five)

### 3. Исследовательский анализ

#### 3.1 Анализ признаков первого заказа
- Распределение пользователей по типам мероприятий, устройствам, регионам
- Влияние характеристик первого заказа на вероятность возврата
- Проверка продуктовых гипотез

#### 3.2 Анализ финансовых показателей
- Связь средней выручки с повторными покупками
- Влияние количества билетов в заказе на лояльность

#### 3.3 Анализ временных характеристик
- Влияние дня недели первой покупки
- Корреляция интервалов между заказами с активностью пользователей

#### 3.4 Корреляционный анализ
- Использование коэффициента корреляции φₖ для анализа связи признаков
- Определение наиболее значимых факторов для количества покупок
