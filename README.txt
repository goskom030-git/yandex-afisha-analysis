# Анализ пользовательского поведения Яндекс Афиши

## Описание
Анализ факторов, влияющих на повторные покупки пользователей.

## Данные
- 290,611 транзакций
- 21,933 пользователей
- Период 2024 год

## Основные выводы
1. 61.4% пользователей совершают повторные покупки
2. Интервал между заказами — ключевой фактор лояльности
3. Desktop-пользователи более лояльны, чем mobile
4. Театральные мероприятия имеют highest retention rate

## Структура проекта
- `analysis.ipynb` — основной анализ
- `requirements.txt` — зависимости
- `.gitignore` — исключаемые файлы
